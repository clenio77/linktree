# Linktree — Digital Card (PT / EN)

Bilingual personal digital card for **Clênio A. Oliveira Moura**.

**Live:** https://clenio77.github.io/linktree/

## Features

- Portuguese / English language switch (saved in `localStorage`)
- Profile, social links, project links, and contact CTAs
- Page QR Code for sharing
- Basic SEO meta tags and Google Analytics events

## Local preview

Serve the folder over HTTP (required for loading `i18n/*.json`):

```bash
python3 -m http.server 8080
```

Open http://localhost:8080

## Content

Edit translations in:

- `i18n/pt.json`
- `i18n/en.json`
